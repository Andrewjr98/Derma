## Description
Derma is a comment app like Reddit but itâ€™s main focus is skincare and makeup. We wanted to make an app where the user felt free to express their opinions and give their honest reviews on different makeup/ skincare products.

## User Story

```

I WANT to be able to create an account and see the reviews
SO I can pick what products work best for me.
```
